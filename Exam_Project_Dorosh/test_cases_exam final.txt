5. TC005_addAndDeleteItemFromCart
Adding and deleting item from the Cart

Steps

1.Open Home page https://thelace.com.ua
2.Click on hamburger box menu on the left in Home page
3.Click on New In products in hamburger menu
4.Check user is redirected to New In page with url  = https://thelace.com.ua/product-category/novynky/
5.Click on any product (second on the page), remember its name
6.Check we were redirected to selected product page
7.Check if name of product on Product Page contains name of product selected on New In Page
8.Click “Add to basket” button
9.Click on button “Cart” in the header of Home Page
10.Click on button “View basket” in Cart Preview Page
11.Check we are redirected to Cart Page with url = https://thelace.com.ua/cart/
12.Check Cart Page has a title – "КОШИК"
13.Check if name of product on Cart Page contains name of product selected on New In Page
14.Click button “Empty Cart”

Expected result

16.Check there is text “Ваш кошик порожній.” on Cart Page
